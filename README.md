### Getting Started

Clone the project repository

```
[https://github.com/LucasAnP/ToDoList.git](https://github.com/LucasAnP/ToDoList.git)
```

Access the project directory

```
cd ToDoList
```

Install the dependencies

```
yarn
  or
npm install
```

Start the APP

```
yarn android
  or
npx expo start

```
